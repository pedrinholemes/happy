# Happy · Backend

> Leve a felicidade para o mundo | Visite orfanatos e mude o dia de muitas crianças.

## Techs

- [x] Typescript
- [x] Express
- [x] TypeORM
- [x] Sqlite3
- [x] Multer
- [x] Yup

## COMANDOS

```sh
## Instale as dependências
yarn install

## Iniciar em modo de desenvolvimento
yarn dev
```
